Task to do.. (a task manager web app)

A simple Trello style task management application built using React.



Features:

* Add, edit, and delete tasks
* Drag and drop tasks between columns
* Responsive layout



Technology used:

* React
* CSS
* HTML5



Structure:

src/

* App.js
* TaskColumn.js
* TaskCard.js
* TaskModal.js
* Tasks.json
* styles.css



Setup Instructions:

* Clone the repository
* npm install
* npm start



Deployment:
The application is deployed using Vercel and is available at:
https://to-do-task-peach.vercel.app/
